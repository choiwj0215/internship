## 프로젝트 소개
기존의 단순 금액 합산 방식의 가계부에서 벗어나, 지출 건별 '필수 여부'와 '주관적 만족도'를 함께 분석합니다. 이를 통해 사용자가 가치 있게 느끼는 소비는 **최대한** 유지하고, 만족도가 낮은 지출(후회 비용)을 타겟팅하여 실현 가능한 다음 달 예산안을 AI가 제안합니다.

---

## 주요 기능

### 1. 지출 데이터 다각도 시각화
* **핵심 지표 요약:** 설정한 목표 예산 대비 총 지출액, 월평균 지출액, 비필수 지출 비율 등.
* **카테고리별 분석:** 카테고리 별 지출 비중(파이 차트)과 순위(바 차트).
* **월별 지출 추이:** 과거 데이터를 함께 업로드할 경우, 월별 지출 흐름을 라인 차트(Line Chart)로 비교 분석이 가능합니다. 

### 2. 가치 소비(만족도) 심층 분석
* **필수 vs 비필수 분석:** 고정 지출과 선택적 지출의 비율을 비교.
* **후회 비용 시각화:** 지출 건별 만족도 분포(Strip chart)와 카테고리별 불만족 집중 구역(Heatmap)을 통해 개선이 필요한 지출 영역을 직관적으로 파악 가능.

### 3. AI 맞춤형 재무 컨설팅 (Github OpenAI API)
* 사이드 바에 이번달 예산을 입력하면, 이를 토대로 지출 점수 표시 ()
* 만족도가 높은 지출은 긍정적 피드백을 제공하고, 만족도 1~2점의 지출 건을 개별적으로 짚어 구체적인 절약 방안을 제시합니다.
* **다음 달 카테고리별 권장 예산안**을 표(Table) 형태로 자동 생성합니다.

### 4. 월간 재무 리포트 자동 생성
* 분석된 핵심 지표, 카테고리 요약, 상위 지출 5건, AI 컨설팅 결과 전문을 제공.

## 사용 방법

1. **데이터 준비:** 분석을 위해 첨부된 `sample_data_2026_01.csv` 파일을 다운로드합니다. (날짜, 카테고리, 내역, 금액, 필수여부, 만족도 컬럼 포함 필수)
2. **웹사이트 접속:** 상단의 Streamlit 배지 링크를 클릭하여 대시보드에 접속합니다.
3. **예산 설정:** 좌측 사이드바에서 이번 달 목표 예산을 입력합니다.
4. **파일 업로드:** * 최상단 업로드 영역의 `1. 이번 달 지출 내역`에 `sample_data_2026_01.csv`를 업로드합니다. **여러 개 데이터를 추가할 수 있으며, 모든 집계,차트와 AI 컨설팅의 대상**
   * (선택) `2. 과거 지출 내역`에 이전 데이터를 업로드하면 월별 추이 비교가 가능합니다. `sample_data_2025_11.csv`,`sample_data_2025_12.csv` **월별 지출 추이에만 적용**
5. **분석 결과 확인:** 하단의 탭을 이동하며 기본 차트와 가치 소비 분석 결과를 확인합니다.
6. **AI 리포트 생성:** `🤖 AI 맞춤 컨설팅` 탭에서 **[AI 재무 리포트 생성하기]** 버튼을 클릭하여 맞춤형 예산안을 확인하고, 리포트를 다운로드합니다.
